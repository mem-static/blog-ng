「CF1264F」Beautiful Fibonacci Problem

September 17, 2020

对于长度为 nnn 的递增等差正整数序列 {a,a+d,a+2d…a+(n−1)d}\{a, a+d, a+2d \ldots a+(n-1)d\}{a,a+d,a+2d…a+(n−1)d}，我们用三元组 (a,d,n)(a,d,n)(a,d,n) 表示。

给定递增等差正整数序列 (a,d,n)(a,d,n)(a,d,n)，你需要构造递增等差正整数序列 (b,e,n)(b,e,n)(b,e,n)，满足：

  • b,e<264b,e < 2^{64}b,e<264，且是正整数
  • 对于所有 0≤i<n0 \leq i < n0≤i<n，a+ida+ida+id 的十进制表示是 Fb+ieF_{b+ie}Fb+ie​ 的十进制表示的后 181818 位的子串（如果没有 181818 位自动补前导零）。其中 FiF_iFi​ 是指斐波那契数列的第 iii 项。

1≤a+(n−1)d≤1061 \leq a+(n-1)d \leq 10^61≤a+(n−1)d≤106。

题解

则有 Fn≡0(modP)F_{n} \equiv 0 \pmod PFn​≡0(modP)，Fn+1≡1(modP)F_{n+1} \equiv 1 \pmod PFn+1​≡1(modP)。考虑：

  • F2n+1≡Fn2+Fn+12≡Fn+12(modP2)F_{2n+1} \equiv F_n^2 + F_{n+1}^2 \equiv F_{n+1}^2 \pmod {P^2}F2n+1​≡Fn2​+Fn+12​≡Fn+12​(modP2)
  • F3n+1≡FnF2n+Fn+1F2n+1≡Fn+13(modP2)F_{3n+1} \equiv F_n F_{2n} + F_{n+1} F_{2n+1} \equiv F_{n+1}^3 \pmod {P^2}F3n+1​≡Fn​F2n​+Fn+1​F2n+1​≡Fn+13​(modP2)
  • …\ldots…
  • Fkn+1≡Fn+1k(modP2)F_{kn+1} \equiv F_{n+1}^k \pmod {P^2}Fkn+1​≡Fn+1k​(modP2)

此例中，我们有 Fn+1≡2tP+1(modP)2F_{n+1} \equiv 2 t P + 1 \pmod P^2Fn+1​≡2tP+1(modP)2，其中 ttt 是与 PPP 互质的整数。

则 Fkn+1≡Fn+1k≡2ktP+1F_{kn+1} \equiv F_{n+1}^k \equiv 2kt P + 1Fkn+1​≡Fn+1k​≡2ktP+1，取 k=5k′t−1k = 5 k' t^{-1}k=5k′t−1，则枚举 1≤k′<1061 \leq k' < 10^61≤k′<106 即可解决本题。

代码（Python）

def exgcd(a,b):
    if b==0:
        return 1,0
    x,y=exgcd(b,a%b)
    return y,x-y*(a//b)
def inv(a,p):
    x,y=exgcd(a,p)
    return (x%p+p)%p
def xmul(a,b):
    c=[[0,0],[0,0]]
    for i in range(0,2):
        for j in range(0,2):
            for k in range(0,2):
                c[i][j]=(c[i][j]+a[i][k]*b[k][j])%(10**12)
    return c
def xpow(a,b):
    s=[[1,0],[0,1]]
    while b:
        if b%2:
            s=xmul(s,a)
        a=xmul(a,a)
        b//=2
    return s
def fib(n):
    if n<=1:
        return n
    return xmul([[1,0],[0,0]],xpow([[1,1],[1,0]],n-1))[0][0]
if __name__=="__main__":
    t=fib(3*(10**6)+1)//(10**6)
